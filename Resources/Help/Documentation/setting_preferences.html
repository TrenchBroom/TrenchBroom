Setting User Preferences

TrenchBroom allows to user to customize some of the functionality of the program. TO do so, open the preferences window with .

General Settings

General Settings

The general settings tab was briefly mentioned in the Getting Started page when setting up Quake's directory. There are more settings here which can be changed.

  • Brightness - This affects the brightness of the textures on the screen. Moving this slider to the right will increase the brightness, making darker textures easier to see.
  • Grid - This slider affects the brightness of the grid lines/checker pattern. Moving it to the right increases the brightness of these marks.
  • Render grid as - This drop down menu lets you choose between grid lines and a checkerboard pattern.
  • Use OpenGL instancing - You should normally never need to touch this, however some videocards do not properly report that they do not support GL_EXT_gpu_shader4 and Trenchbroom may crash when trying to go into vertex editing mode. To resolve this, set this to Forced off.
  • Texture Browser Icon Size - This affects the size of the texture icons in the texture browser pane.
  • Mouse Look - This affects the speed at which the camera rotates with the mouse.
  • Mouse Pan - This affects the speed the camera moves when panning.
  • Mouse Move - This affects the speed the camera moves forward and back when scrolling the mouse wheel.

Keyboard Settings

General Settings

The keyboard settings tab allows you to customize nearly all the shortcuts in Trenchbroom. To do so, click twice the Shortcut in the Shortcut column and then press the new key combination you would like to use. Shortcuts which are used more than once will show up as red and you can scroll through the list to find the other instance of its use.
